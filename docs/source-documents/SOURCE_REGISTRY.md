## Key Source Relationships

- The **VLEP infographic** (The_VLEP_Precision_Neurology_Algorithm.jpeg) visually confirms the 4-layer system: Immutable Evidence Ledger (L), 6-Domain Phenotype Vector P(i), LPA (HMM + GLMMs), and CSEP snapshot. It also confirms the 17.6% evidence survival rate heuristic and the 24-month integration roadmap.
- The **NotebookLM Mind Map** establishes the full project scope tree, including hippocampal long axis (anatomical differentiation, functional gradients, seizure networks), the Epilepsy Phenotype Project core (LPA, system architecture, clinical adoption, technical standards), genetic architecture and comorbidity (sleep-epilepsy axis, pleiotropic candidate genes), and governance/clinical strategy.
- The **LPA Technical Blueprint** (.pptx) provides the presentation-layer specification of the Longitudinal Phenotyping Algorithm.
- The **Bench-to-Bedside document** covers the clinical translation lifecycle and maps research-phase milestones to clinical deployment readiness.
- The **Strategic Implementation Roadmap** covers enterprise-scale deployment architecture, phased rollout, and validation milestones.
- The **Hippocampal Long Axis documents** are foundational neurobiological sources for seizure network modeling within the VLEP framework.
